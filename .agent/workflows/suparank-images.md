# SupaRank Image SEO Audit

Görsel SEO optimizasyonu analizi.

## Kullanım
```
/suparank-images https://example.com
```

## Kontroller

### Alt Text
- [ ] Tüm görsellerde alt attribute var
- [ ] Alt text açıklayıcı ve anlamlı
- [ ] Anahtar kelime içeriyor (doğal şekilde)
- [ ] 125 karakterden kısa

### Dosya Adları
- [ ] Açıklayıcı dosya adları (resim1.jpg ❌ → bilsem-muzik-testi.jpg ✅)
- [ ] Küçük harf kullanımı
- [ ] Tire ile ayrılmış kelimeler

### Format ve Boyut
- [ ] Modern formatlar (WebP, AVIF)
- [ ] Uygun sıkıştırma
- [ ] Responsive görseller (srcset)
- [ ] Boyut belirtilmiş (width/height)

### Performans
- [ ] Lazy loading aktif
- [ ] CDN kullanımı
- [ ] Önbellekleme

### Erişilebilirlik
- [ ] Dekoratif görsellerde alt="" (boş)
- [ ] Karmaşık görsellerde figcaption

## Optimizasyon Önerileri
Her sorun için specific düzeltme önerileri sun.
