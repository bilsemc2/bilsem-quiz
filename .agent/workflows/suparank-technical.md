# SupaRank Technical SEO Audit

Teknik SEO faktörlerini analiz eder.

## Kullanım
```
/suparank-technical https://example.com
```

## Kontroller

### Crawlability
- [ ] robots.txt doğru yapılandırılmış
- [ ] Sitemap.xml mevcut ve geçerli
- [ ] Canonical URL'ler tanımlı
- [ ] Noindex/nofollow kontrolleri

### Indexing
- [ ] Index durumu
- [ ] Duplicate content kontrolü
- [ ] Thin content kontrolü

### Site Yapısı
- [ ] URL yapısı düzgün
- [ ] Breadcrumb navigasyonu
- [ ] Internal linking yapısı
- [ ] Orphan sayfa kontrolü

### Güvenlik
- [ ] HTTPS aktif
- [ ] Mixed content yok
- [ ] HSTS header'ı

### Performans
- [ ] Gzip/Brotli sıkıştırma
- [ ] Cache headers
- [ ] CDN kullanımı

### Mobil
- [ ] Mobile-friendly
- [ ] Viewport ayarları
- [ ] Touch target boyutları
