# SupaRank On-Page SEO Audit

Sayfa içi SEO faktörlerini analiz eder.

## Kullanım
```
/suparank-onpage https://example.com
```

## Kontrol Listesi (25+ Kontrol)

### İçerik Kalitesi
- [ ] Minimum 300 kelime içerik
- [ ] Anahtar kelime yoğunluğu (%1-3)
- [ ] İlk 100 kelimede anahtar kelime
- [ ] LSI (ilgili) kelimeler kullanımı

### Heading Yapısı
- [ ] Tek H1 tag'i
- [ ] H1'de anahtar kelime
- [ ] H2-H6 hiyerarşisi doğru
- [ ] Her H2 altında içerik var

### URL Yapısı
- [ ] Kısa ve açıklayıcı URL
- [ ] Anahtar kelime içeriyor
- [ ] Küçük harf kullanımı
- [ ] Tire (-) ile ayrılmış kelimeler

### Görseller
- [ ] Alt text mevcut
- [ ] Dosya adları açıklayıcı
- [ ] Boyutlar optimize
- [ ] Lazy loading aktif

### İç Linkler
- [ ] İlgili sayfalara linkler
- [ ] Anchor text anlamlı
- [ ] Kırık link yok
- [ ] Ortalama 3-5 iç link

### Mobil Uyumluluk
- [ ] Responsive tasarım
- [ ] Touch-friendly elementler
- [ ] Viewport meta tag

### Sayfa Hızı
- [ ] CSS/JS minify
- [ ] Görsel optimizasyonu
- [ ] Lazy loading
- [ ] Caching aktif
